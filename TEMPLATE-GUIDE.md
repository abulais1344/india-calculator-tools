## 📝 How to Customize

### Option A: Edit HTML Directly
File location: `business-proposal-template.html`

Changes you can make:
- Update text in `<p>` tags
- Modify prices and timelines
- Change service descriptions
- Edit payment terms
- Customize section content

### Option B: Modify Colors
Look for `:root` section in CSS:
```css
--primary: #2563eb;      /* Change main color */
--accent: #f59e0b;       /* Change accent color */
--text-dark: #1f2937;    /* Change text color */
```

### Option C: Regenerate in Canva
1. Open your template link
2. Take screenshots of each section
3. Recreate design in Canva using screenshots as reference
4. Export as Canva link or PDF
