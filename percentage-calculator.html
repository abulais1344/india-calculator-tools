What Is X% of Y?

Result-

Formula: Result = (X ÷ 100) × Y

Example: 18% of ₹2500 = (18 ÷ 100) × 2500 = 0.18 × 2500 = ₹450. Useful for GST approximation, discounts, and commission calculations.

Percentage Increase / Decrease

Formula: Percentage Change = ((New - Old) ÷ Old) × 100

Example: Old = ₹800, New = ₹1000. Change = ((1000 - 800) ÷ 800) × 100 = (200 ÷ 800) × 100 = 25% increase. Useful for salary hikes, sales growth, stock movement, and inflation tracking.

Frequently Asked Questions About Percentages

What is the easiest way to calculate a percentage mentally?

For 10%, divide by 10. For 5%, take 10% and divide by 2. For 15%, add 10% and 5%. For 20%, divide by 5. Practice these shortcuts to calculate percentages quickly in your head without a calculator.

How do I calculate the original value if I know the percentage increase?

If the new value is 125% of the original (25% increase), divide new value by 1.25 to get the original. Formula: Original = New ÷ (1 + Percentage Increase ÷ 100).

Can percentage increase and decrease be treated the same way?

No. A 50% increase is NOT reversed by a 50% decrease. For example, ₹100 increased by 50% = ₹150. Then ₹150 decreased by 50% = ₹75, not ₹100. This is because the percentage is applied to different base values.
